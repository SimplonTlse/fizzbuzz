FizzBuzz

Modifiez le fichier fizzbuzz.js en écrivant un algorithme à l'intérieur de la fonction qui :

  1. remplace les multiples de 3 et 5 par la chaîne de caractère fizzbuzz
  2. remplace les multiples de 3 par la chaîne de caractère fizz
  3. remplace les multiples de 5 par la chaîne de caractère buzz
  4. renvoie toutes les autres valeurs sous forme d'entier

Pour valider votre code, vous devez en ligne de commande dans le dossier du projet exécuter la commande mocha si tous les tests passent au vert, good job !

Pour installer Mocha : npm install -g mocha

Bonus mode :

  • N'utilisez pas de else
